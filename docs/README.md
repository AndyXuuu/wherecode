# Docs 说明

本目录存放架构、协议与运行手册文档，作为前后端和 Agent 协作基线。

- `architecture.md`: 系统分层与模块边界
- `protocol.md`: 客户端/服务端消息协议
- `runbook.md`: 本地运行、联调与故障排查流程
- `data_model.md`: 项目 -> 任务 -> 命令的数据结构
- `command_center_pencil_spec.md`: Command Center 的 Pencil 拆解与实现规范
- `design_acceptance.md`: Pencil 设计验收记录（只读检查与结论）
- `pencil_app_pages_mapping.md`: Pencil 节点到 Next 路由/组件的映射清单
- `progress_audit_2026-02-22.md`: 项目进度审计结论与下一阶段建议
